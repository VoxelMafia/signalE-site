# SignalE — impact site

Public landing page for **SignalE**, an AI energy system manager for Windows that quietly
cuts wasted PC power and shifts flexible work to cleaner grid hours — with the value pooled
into a public wealth fund rather than scattered as tiny payouts.

**Live:** https://voxelmafia.github.io/signalE-site/

This repository hosts only the static **impact dashboard** ([`index.html`](index.html)) via
GitHub Pages. It's a self-contained page — the interactive figures are computed in-browser
from the same Monte-Carlo model that lives in the (currently private) SignalE code repo.

## Status

SignalE is an **early technical preview**. The runnable prototype, full design, impact/business
models, and the DreamGoal campaign assets live in a separate repository; code and downloads will
be published when the preview is ready for broader use.

## The numbers, honestly

Every figure on the page is a **modeled scenario** (median of 100k Monte-Carlo trials), annual,
in-use only, and net of SignalE's own overhead — not a balance already achieved today. Assumptions
are conservative and sourced (IEA/Ember grid intensity, EPA equivalencies, ~1.4B active Windows
devices). No peak-as-average, no greenwashing.
